# Contador_Bin-rio_ESP32S3
Projeto voltado a desenvolver um contador binário em um microcontrolador ESP32S3

um programa utilizando o ESP-IDF para implementar um contador binário de 4 bits. O valor atual do contador deve ser exibido utilizando 4 LEDs. Além disso, o sistema deve utilizar 2 botões conectados a entradas digitais, com as seguintes funcionalidades:
Botão A: a cada acionamento, deve incrementar o valor do contador conforme a unidade de incremento atual (padrão: +1).


Botão B: alterna a unidade de incremento entre +1 e +2 a cada acionamento.
